## Instalation

If you haven't a rust environment you can use [rustup](https://rustup.rs/) or your package manager.

### With cargo

Recommended solution.

```
cargo install --git https://github.com/natir/br.git
```

### With source

```
git clone https://github.com/natir/br.git
cd br
cargo install --path .
```
